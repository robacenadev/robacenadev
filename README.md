## 📌 Projeto em Destaque

### 🌐 Plataforma de Gestão de Custos Multi-Cloud

> Plataforma web corporativa para **consolidação, análise e gestão de custos** em ambientes AWS, Azure e Huawei Cloud.

Assumi o projeto após a saída do desenvolvedor anterior, atuando diretamente em toda a stack:

| Área | Atividades |
|------|-----------|
| **Backend** | Manutenção e evolução em Django, refatoração de código legado |
| **Automação** | Pipelines de ingestão de dados de billing via n8n |
| **Banco de Dados** | Estruturação e otimização de MySQL |
| **Frontend** | Dashboards com Chart.js, exportação de relatórios PDF |
| **Infra** | Docker Swarm + Traefik, multi-cliente com segregação de dados |

`Python` `Django` `MySQL` `n8n` `Docker Swarm` `Traefik` `JavaScript` `Chart.js`
